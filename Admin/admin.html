Access Levels

Different permission levels for various administrative roles

Super Administrator

  • Full system access and control
  • User management and role assignment
  • System configuration and settings
  • Security and audit management
  • Database administration

Administrator

  • Content management and updates
  • Message and feedback management
  • Event and announcement posting
  • Basic user account management
  • Analytics and reporting access

Faculty Administrator

  • Faculty profile management
  • Academic content updates
  • Student inquiry responses
  • Course and program information
  • Limited reporting access

Department Administrator

  • Department-specific content
  • Faculty information within department
  • Department event management
  • Student messages for department
  • Department analytics only
